# multiple-choice-test
An application which helps you to make a multiple-choice test

## How to install
1. **Windows:** run `*.exe` file directly
2. **Linux:** execute `*.py` file
    - Used package: `tkinter`, `datetime`.
    - Use Python 3 to make the code.
